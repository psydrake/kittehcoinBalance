FAQ

  • Why does my balance in KittehCoin Balance look lower than the balance shown in my wallet?

    KittehCoin Balance examines the literal addresses you enter and reports the balances from the blockchain explorer. However, I have found that if I send coins from my QT wallet to another address, the wallet can generate a new internal address that it can shuffle coins into, around the same time that it sends the coins to the external address.

    Basically this means that if your balance in KittehCoin Balance looks off, it is probably because some of your balance has been moved to a hidden address in your wallet - in my experience, as a result of an attempted send transaction.

    You can find the hidden address(es) by following these instructions for the QT wallet:

    1. Go to Help -> Debug Wallet in the Wallet menu
    2. Click on the Console tab
    3. There should be a command prompt at the bottom of the window. Enter: listaddressgroupings and press RETURN
    4. It should give you a list of addresses and balances - for example: 
      	[
		[
			["KNNh6DNUZuxZP2HLTRK812gVC3bJoyVaFJ", 500.00000000, "Tips"]
		],
		[
			["KJeiba3V4w52Dn7sceMaZQF1vtnSDbU9gG", 1002.72180769],
			["K7M4S6Jq4A1rs7QbNJw1SLTxKAAmhd2Qjj", 192916.32679533, "Mining"]
		],
		[
			["KDbtYRiPd6sG7F8oaK2XSLJdJFuDnpGD5d", 60276.66993597, "Donations"]
		]
	]
      Notice that there is an address, KJeiba3V4w52Dn7sceMaZQF1vtnSDbU9gG in this case (yours will be different), that was created by the wallet without a label - probably during a send operation.
    5. Add that address into the KittehCoin Balance app. The balance shown should now match that shown in your KittehCoin QT wallet.

    Because of this wallet behavior, I've found it more practical to use KittehCoin Balance with addresses from a wallet from which I don't do a lot of sending - e.g. the wallet that holds my mining address from a mining pool, or a buy-and-hold address that I use to withdraw from a currency exchange - as opposed to a wallet from which I'm doing active sending of funds.
